Retinizarea imaginilor

Cel mai simplu mod de a implementa receptivitatea este de a folosi o singură imagine pentru toate dispozitivele, atribuindu-i un anumit set de proprietăți CSS. /* Proprietățile ar trebui să fie aplicate tuturor imaginilor, de aceea se folosește selectorul de tag. */ .responsive-img { display: block; max-width: 100%; height: auto; }

Această abordare de bază a imaginilor de conținut este prea simplă și nu rezolvă problema ecranelor Retina și a dimensiunii imaginilor mari de pe dispozitivele mobile, dar este un început bun.
