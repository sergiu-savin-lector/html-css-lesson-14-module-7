Tag-ul <picture>

Elementul HTML <picture> conține zero sau mai multe elemente <source> și un element <img> pentru a oferi versiuni alternative ale unei imagini pentru diferite scenarii de afișare/dispozitiv.

Browserul va lua în considerare fiecare element <source> copil și va alege cea mai bună potrivire dintre ele. Dacă nu sunt găsite potriviri - sau browserul nu acceptă elementul <picture> - este selectată adresa URL a atributului src al elementului <img>. Imaginea selectată este apoi prezentată în spațiul ocupat de elementul <img>.

<picture> <source srcset="image.avif" type="image/avif"> <source srcset="image.webp" type="image/webp"> <img src="image.jpg" alt="A description of the image." width="300" height="200" loading="lazy" decoding="async"> </picture>

Primul element sursă indică o imagine în noul format AVIF. Dacă browserul este capabil să redea imagini AVIF, atunci acesta este fișierul imagine pe care îl alege. În caz contrar, se trece la următorul element sursă.

Al doilea element sursă indică o imagine în format WebP. Dacă browserul este capabil să redea imagini WebP, va folosi acel fișier imagine.

În caz contrar, browserul va reveni la fișierul imagine din atributul src al elementului img. Această imagine este un JPEG.

Cat timp nu avem nevoie de alt format, putem sa le folosim oricare dintre cele doua variante:

Fie tag img cu atribut srcset <img srcset="foo-320w.jpg 320w, foo-480w.jpg 480w, foo-800w.jpg 800w" sizes="(max-width: 320px) 280px, (max-width: 480px) 440px, 800px" src="foo-800w.jpg" alt="bar" />

Fie tag picture, cu elemente source si atribute srcset <picture> <source media="(max-width: 799px)" srcset="foo-480w.jpg" /> <source media="(min-width: 800px)" srcset="foo-800w.jpg" /> <img src="foo-800w.jpg" alt="bar" /> </picture>
